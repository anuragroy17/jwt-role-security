# JWT-Integration Role Based - A Demo
JWT Intergration in Spring Boot(Maven Multi Module) and Angular 8 with Role Based Support -

 - Added JWT token security (lasts one hour) on both frontend and backend
 - Added AuthGuard to router links for unauthorized access to urls
 - Backend has support for 'hasRole' for every mapping
